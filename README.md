# OpenFCS

Open Financial Crime Systems Lab (OpenFCS) is a product-grade, transparent financial crime monitoring simulation platform.

## Vision
To build an open, web-based financial crime monitoring system demonstrating:
- Deterministic rule-based enforcement
- Probabilistic ML prioritization
- Operational AML tradeoffs
- Enterprise-grade architecture

## Bank Simulation
Midland Regional Bank (MRB)
- 50,000 customers
- 1.2M annual transactions
- Retail & SME focus
- 6 core AML monitoring rules
- ML prioritization layer

## Architecture Layers
1. Data Simulation Layer
2. Deterministic Monitoring Engine
3. ML Prioritization Layer
4. Case Management Engine
5. Analytics & Metrics Engine
